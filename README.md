# victoria-hospital-app

## Structure

- `frontend/` React Native (Expo)
- `backend/` Node.js + Express

## Frontend

```bash
cd frontend
npm install
npm start
```

Or from the repo root:

```bash
npm run frontend:start
```

## Backend

```bash
cd backend
npm install
npm run dev
```

Or from the repo root:

```bash
npm run backend:dev
```

Copy `backend/.env.example` to `backend/.env` and fill values (don’t commit secrets).
